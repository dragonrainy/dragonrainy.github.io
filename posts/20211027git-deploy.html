git部署本地文件到仓库

Created2021-10-27|Updated2021-10-27
|Post View:

尚未建立仓库

安装Git

参见廖雪峰老师Git教程

新建仓库

注册/登录Github，新建仓库。

SSH Key

  1. 部署全局账号，按如下指令输入github账号名称和邮件
  2. 输入指令ssh-keygen -t rsa -C "account@email.com，引号内为注册Github所使用的邮箱，按照提示生成共钥文件id_rsa.pub，默认路径在/c/Users/Username/.ssh文件夹下，也可以自定义储存路径，用文本编辑器打开，复制公钥字符串。
  3. 登录github。打开setting->SSH keys，点击右上角 New SSH key，把生复制的公钥id_rsa.pub的信息粘贴到key输入框中，再为当前的key起一个title来区分每个key。
  4. 输入指令ssh -T git@github.com检查是否连接成功，未成功可能出现以下结果 输入yes直到出现 则说明连接成功

    生成和部署

    由于版本变化命令可能有所更新，新生成仓库后会出现如下提示，其它博客教程由于时间较早而和新版有出入，请以新版为准。
  5. git init新生成.git文件夹
  6. git add .将所有文件添加到临时区域
  7. git commit -m "first commit"引号内为提交的备注信息
  8. git branch -M main
  9. git remote add origin git@github.com:Toby404502/Toby404502.github.io.gitorigin后面为仓库的ssh地址
  10. git push -u origin main部署到远程仓库
